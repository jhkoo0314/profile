Behavior Definition 09

HIR (High-Impact Rate): '유효 행동' 중심의 효율성 평가

Efficiency Metric

단순 반복 업무(Low Impact)와 전략적 핵심 업무(High Impact)를 구분하여, 성과에 직결되는 행동 비율을 측정합니다.

HIR = Σ ( 행동가중치 × 질적점수 × 영향력계수 ) / 총활동수

• 행동가중치: PT(3.0) > 디테일(2.0) > 해피콜(1.0)

• 질적점수: 체류시간, 자료 활용 등 보정 (0.8~1.2)

• 영향력계수: 매출 상관관계 분석 가산점

💡 기대 효과:
영업사원이 실적을 채우기 위해 의미 없는 '해피콜'만 반복하는 허수 영업(Abuse)을 방지합니다.

Key Logic #1: HIR 10

RTR (Relationship Temperature): 고객 관계의 '유효기간' 관리

Health Metric

고객과의 관계를 '온도'로 시각화합니다. 마지막 방문 후 시간이 지날수록 온도가 내려가며(Time Decay), 이를 통해 이탈을 방지합니다.

RTR = Σ ( Sentiment × Time Decay )
function calculateDecay(days) {
  if (days <= 7) return 1.0; // Hot
  if (days <= 30) return 0.5; // Warm
  return 0.1; // Cold (Churn Risk)
}

Key Logic #2: RTR 11

BCR (Behavior Consistency): 꾸준한 영업 활동을 위한 루틴관리

Routine Metric

영업 활동의 총량뿐만 아니라 '규칙성'을 평가합니다. 표준편차를 활용하여 비정기적 '활동 편중(Activity Bias)'을 방지하고 예측 가능성을 높입니다.

BCR = (Total Activity × 0.4) + (1 / SD of Visit Interval × 0.6)
* 표준편차(SD)가 낮을수록(=꾸준할수록) 점수가 높아지는 역수 구조

// Normalization Implementation (Z-Score)
function normalize(score, mean, std) {
    const z = (score - mean) / (std || 1);
    // T-Score conversion for 0-100 scale
    return Math.min(100, Math.max(0, (z * 10) + 70));
}
