Project 01. Behavior-Driven CRM v2.1

Core Concept: 행동 데이터의 자산화

  • Quality Metrics: 방문 횟수(Quantity) 중심의 관리를 8단계 핵심 영업 행동의 질적 평가(Quality) 체계로 전환
  • 3-Layer Engine: 행동(Behavior) → 분석(Analytics) → 성과(Outcome)를 연결하는 데이터 파이프라인 설계
  • SFE Modeling: 고성과자의 승리 공식(Winning Pattern)을 추출하여 조직 구성원의 상향 평준화를 위한 행동 모델링

Leading Indicators (선행 지표)

  • HIR (Impact): 성과 직결 행동 비중 측정
  • RTR (Relationship): Time Decay 로직 기반 고객 관계 온도 시각화 및 이탈 위험 예측
  • BCR (Consistency): 활동 편중 방지를 위한 규칙성 평가
  • PI (Index): 병원 규모별 난이도 보정 지수

Strategic Impact: 기획의 가치

매출이라는 후행 지표 이전에 성공 확률을 예측할 수 있는 대시보드(Coaching Cockpit)를 구축하여, 관리자가 정확한 숫자를 가지고 구체적인 행동 교정을 지시할 수 있는 체계를 완성했습니다.

Case Study: BIDIX

"BIDIX는 부동산 권리 분석을 어떻게 하면 쉽게 할 수 있을까하는 문제인식에서 시작했습니다."
무수한 변수와 법적 리스크가 산재한 부동산 경매 도메인을 하나의 유기적인 전략 맵으로 설계했으며, 주관적 판단을 배제한 객관적인 알고리즘으로 만들어 무결성을 증명한 프로젝트입니다.

Project 02. Strategic Planning & System Building

Decision Pipeline: 판단의 표준화

  • Business Modeling
    18종의 복잡한 권리 관계와 시장 변수를 분석 프레임워크로 체계화하여 리스크 구조 장악
  • Strategy Operating Framework
    [데이터 추출 → 전략 대입 → 성과 산출]의 3단계 표준 프로세스 정립
  • Automated Compliance
    실시간 법령 참조를 통해 정책 위반 리스크를 원천 차단하는 지능형 운영 환경 구축

Core Engine: 데이터 자산화 및 판단 모델

  • Court Data Normalization
    법원의 비정형 공시 데이터를 수집하고, 분석 가능한 표준 데이터셋(Standard DB)으로 정규화
  • Legal Judgment Model
    경매 권리 분석 로직을 단일 원칙으로 구현하여 판단의 일관성 및 정밀도 확보
  • Logic Reliability
    수천 개의 시뮬레이션을 통해 설계된 비즈니스 규칙의 무결성과 실전 작동 여부 검증

Model Reliability: 전략의 무결성 증명

BIDIX 프로젝트는 시뮬레이션을 통해 설계된 비즈니스 모델이 현장에서 작동함을 검증했으며, 이론적인 기획에서 멈추지 않고 실제 시스템으로 구현할 수 있는 실행력을 보여주는 프로젝트입니다.
