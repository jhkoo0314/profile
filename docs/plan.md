Project: The Sales Architect’s Hyper-Cube
Concept: Deconstruct & Reconstruct (영업의 해체와 재구축)
Visual Theme: Dark Sci-Fi + Glassmorphism + Neon Data
🏗️ 3D Visual & Motion Blueprint (Section-by-Section)
📍 Scene 1: The Black Box (Hero Section)
Visual Concept: "영업이라는 미지의 영역(The Black Box)"
3D Object: 화면 정중앙에 거대한, 검은색 무광의 **금속 큐브(Obsidian Cube)**가 공중에 떠서 천천히 회전하고 있습니다. 표면은 거칠지만, 틈새 사이로 푸른빛(Data Light)이 새어 나옵니다.
Interaction Logic:
User Action: 스크롤을 시작(0% -> 10%)하는 순간.
3D Reaction: 큐브가 Explode(폭발/분해)합니다. 큐브의 껍질이 사방으로 흩어지며, 그 안에서 복잡하지만 정교하게 돌아가는 **황금빛 기계 장치(The Engine: Sales Logic)**가 드러납니다.
Camera: 큐브 외부에서 내부 코어(Core)로 줌인(Zoom-in)하며 빨려 들어가는 카메라 워킹.
Copywriting (Floating 3D Text):
(Before Scroll) "Sales is a Black Box."
(After Scroll) "I decoded it."
📍 Scene 2: The Data Tunnel (Philosophy)
Visual Concept: "시간의 터널을 통과하는 경험"
3D Object: 구재현 님의 14년 커리어가 평면적인 타임라인이 아닌, '데이터 터널(Data Tunnel)' 형태로 구현됩니다.
Interaction Logic:
User Action: 계속 스크롤 (Scroll Loop).
3D Reaction: 사용자의 시점(Camera)이 끝없는 터널을 비행합니다.
왼쪽 벽면: GC녹십자, 대학병원, 현장 영업 등의 키워드가 거대한 홀로그램 빌딩처럼 지나갑니다.
오른쪽 벽면: 해당 시기의 성과 데이터(M/S 1위, KPI 달성)가 그래프 형태로 솟아오릅니다.
Highlight: 2022~2026년 구간(공백기/R&D)에 진입하면, 터널의 조명이 **붉은색(Red)**에서 **청록색(Cyan)**으로 바뀌며 속도가 빨라집니다(Warp Drive 효과).
Micro-Interaction: 마우스를 움직이면 터널의 벽면이 커서 방향으로 살짝 기울어지는(Parallax) 효과.
📍 Scene 3: The Architecture (BIDIX Case Study)
Visual Concept: "설계도(Blueprint) 위에서 작동하는 시뮬레이션"
3D Object: 반투명한 아이소메트릭(Isometric) 3D 도시가 나타납니다. 각 건물은 '고객(Hospital/Client)'을 상징합니다.
Interaction Logic:
User Action: 스크롤 시 '문제 해결' 단계 진행.
Phase 1 (Scan): 도시 위로 레이저 스캐너가 지나가며 위험한 건물(Risk)은 빨간색, 유망한 건물(Opportunity)은 초록색으로 칠해집니다. (HIR/RTR 로직 시각화)
Phase 2 (Optimize): 도시의 건물들이 재배치되며 최적의 경로(Sales Route)가 빛나는 선으로 연결됩니다.
Phase 3 (Growth): 도시의 건물 높이가 실시간으로 솟아오르며(Scale Up), 화면 중앙에 **"Growth +2.5x"**라는 3D 텍스트가 강하게 충돌하며(Impact) 멈춥니다.
📍 Scene 4: The DNA Helix (Synergy)
Visual Concept: "현장과 기술의 결합"
3D Object: 두 개의 서로 다른 물질이 나선형으로 꼬여있는 DNA 구조.
물질 A (영업): 거친 돌 질감 (Rough Stone) - 현장의 단단함.
물질 B (기술): 액체 금속 질감 (Liquid Mercury) - 기술의 유연함.
Interaction Logic:
스크롤에 따라 두 물질이 서로 융합되어, 마침내 하나의 완벽하고 매끄러운 다이아몬드(Crystal) 형태로 변환됩니다.
변환된 크리스털 안에서 **"Strategic Sales Planner"**라는 텍스트가 굴절되어 보입니다.
📍 Scene 5: The Holographic Card (Contact)
Visual Concept: "미래에서 온 명함"
3D Object: 화면에 얇은 **유리 명함(Glass Card)**이 둥둥 떠 있습니다.
Interaction Logic:
Mouse Move: 마우스 위치에 따라 명함이 기울어지며(Tilt), 표면에 빛 반사(Specular Highlight)가 흐릅니다.
Hover: 이메일 주소 위에 마우스를 올리면, 명함이 180도 뒤집히며 뒷면의 "Send Message" 버튼이 활성화됩니다.
Background: 배경에는 희미한 입자(Particle)들이 구재현 님의 얼굴 형상이나 데이터 클라우드 형상을 만들었다 흩어지기를 반복합니다.
